Misleading Graph Examples

Wrong points

  1. 3D graph may not show the accurate ratio between data
  2. side perspective will exaggerated the closer data
  3. no indicators on y-axis -> cannot show the exact data to readers

Ways to improve

  • use 2D graph
  • shown the graph in front view instead of side
  • indicate the amount on y-axis, the number should start at zero

Wrong points

  1. Not start at zero
  2. line graph should be used to show trend
  3. dull when using 2D (but good to be more accurate)

Ways to improve

  • y-axis should start at zero
  • choose the correct graph type, know the characteristic of different graph
